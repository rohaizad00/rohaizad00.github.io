Directory & Assets System

System Overview

This system allows users to view staff details and track assigned assets. It is designed to manage staff information efficiently and ensure accountability for company assets.

Key Features

Staff Information Management
  • Display comprehensive staff details to users.
  • Admins can add, edit, delete, and update staff information.
Asset Management
  • IT personnel can assign assets to staff.
  • The system tracks assets assigned to each staff member.

User Roles

  • User: Can view staff information.
  • Admin: Can manage staff information (add/edit/delete/update).
  • IT: Can assign and track assets for staff.

Figma Design Focus

  • Dashboard showing staff list with search and filter options.
  • Forms for admin to manage staff data.
  • Asset assignment interface for IT personnel with tracking features.
